## Objective
Design of a folded cascode op-amp targeting >70 dB gain and >50° phase margin using 180nm CMOS process.

## Tools Used
- Cadence Virtuoso
- Spectre Simulator

## Highlights
- Fully differential schematic
- Manual layout with DRC and LVS clean
- Simulated gain: 75.3 dB
- Phase margin: 58°
